# Lender Dashboard – Design Specs

## Figma Link
https://figma.com/your-link

## Overview
A single-page lender dashboard designed for institutional users to monitor portfolio performance, risk, and returns.

## Layout Structure
- KPI cards (Portfolio, Returns, Risk, Active Loans)
- Active loans table
- Charts (returns over time, risk distribution)
- Action buttons (Deposit, Withdraw)

## Components
- KPI Card
- Data Table
- Chart (Line, Pie)
- Buttons (Primary, Secondary)
- Filters & Dropdowns

## Design System
- TailwindCSS + ShadCN
- Dark Theme (#0B0A1D background, #1F1C3D primary)

## Responsiveness
- Desktop: 12-column grid
- Mobile: stacked cards and collapsible table

## Notes
- Focus on clarity and data density
- Institutional-grade UI
